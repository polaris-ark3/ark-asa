Steampunk Ascent - Пакет расширения для ARK: Survival Ascended

**Steampunk Ascent** — это платное расширение для **ARK: Survival Ascended**, выпущенное вместе с бесплатным дополнением **Aberration DLC**. Оно является частью пакета **Bob's Tall Tales DLC** и добавляет новые скины, структуры, предметы и существ в игру.

Содержание

1. Косметические предметы

  • Косметика для шлемов: Берет, Шляпа лодочника, Шлем велосипедиста, Шлем инженера, Элегантная шляпа
  • Косметика для создания седла: Стимпанковое седло для Rock Drake
  • Стимпанковая шляпа и очки
  • Стимпанковые очки

2. Существа

  • Cosmo - новое существо в мире ARK
  • Sir-5rM8 - механическое существо

3. Структуры

  • Часы
  • Инкубатор эмбрионов
  • Хранение генов
  • Промышленный контейнер для консервирования
  • Стеллаж для книг
  • Соединённые коробки для хранения
  • Паровая кузня
  • Прожектор
  • Тесла-коил
  • Стол для механики
  • Цеппелин

4. Предметы

  • Пульсатор Cosmo
  • Генетический сканер
  • Пистолет для геля
  • Таймер
