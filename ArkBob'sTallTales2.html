Westland War - Пакет расширения для ARK: Survival Ascended (DLC Extinction)

**Westland War** — это уникальное расширение для **DLC Extinction** в игре **ARK: Survival Ascended**. Оно добавляет элементы в стиле Дикого Запада, включая новые существа, структуры и предметы, которые позволяют игрокам погрузиться в атмосферу войны и борьбы на постапокалиптическом Востоке.

Содержание пакета

1. Косметические предметы

  • Шляпа ковбоя
  • Кожаная броня в стиле Дикого Запада
  • Рубашка и джинсы в стиле вестерн
  • Стимпанковая шляпа и очки

2. Существа

  • Battle Rig - боевой механизм, используемый для битв в стиле "Дикого Запада".
  • Westland T-Rex - улучшенная версия Тираннозавра, с броней и усиленной атакой.
  • Mech-Shooter - механическое существо с пулеметом для защиты территории.

3. Структуры

  • Боевой шатер
  • Заправочная станция для мехов
  • Ремонтные ангар для боевых механизмов
  • Склад для боевых снарядов
  • Трансформатор для энергетических ресурсов

4. Предметы

  • Динамит - для разрушения баррикад и создания новых путей.
  • Реактор энергии - для питания боевых механизмов и силовых установок.
  • Мех-пулемет - оружие для защиты и нападения в битвах.
