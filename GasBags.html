Газбагс (Gasbags) - Существо из DLC Extinction

**Газбагс** - это уникальное существо, представленное в дополнении **Extinction DLC** для **ARK: Survival Evolved**. Газбагс — это огромный воздушный животное, которое использует газовые мешки в своем теле, чтобы парить в воздухе и двигаться по карте. Это существо идеально подходит для транспортировки и борьбы в условиях Extinction.

Досье: Газбагс

Газбагс — это мирное существо, которое может быть приручено с помощью пищи. С помощью своей уникальной способности накапливать газы, Газбагс способен подняться в воздух и летать. Это существо может быть использовано для транспортировки больших грузов и даже в бою, когда оно накапливает достаточно газа для резкого рывка в воздух.

Факты о Газбагс:

  • Тип: Воздушное существо
  • Размер: Большое
  • Особенность: Парит в воздухе, используя газовые мешки
  • Приручение: С помощью пищи (овощи и фрукты)
  • Использование: Транспортировка, исследование, бой
  • Способности: Газбагс может использовать накопленные газы для быстрого подъема в воздух, а также для преодоления препятствий.

Приземление Газбагс:

Газбагс может быть приземлен, если его газовые мешки опустошены, что позволяет ему безопасно приземлиться и продолжить свою задачу. Он также может быть использован в боевых ситуациях, где его способность к перемещению через воздух может быть решающим фактором.
